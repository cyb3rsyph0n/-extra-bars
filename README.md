# ExtraBars

**Customizable action bars for consumables, abilities, and more!**

ExtraBars lets you create additional action bars specifically designed for tracking and using your consumables, racial abilities, trinkets, and other items you want quick access to.

## Features

- **Multiple Custom Bars** - Create as many bars as you need for different purposes
- **Smart Categories** - Automatically populates with items you have:
  - Racial Abilities
  - Food & Drink
  - Health Potions (includes Healthstones!)
  - Flasks & Phials
  - Combat Potions
  - Equipped Trinkets (on-use only)
- **Inventory Browser** - Add any on-use item from your bags directly to your bars
- **Persistent Custom Items** - Items you add from inventory stay on your bar even when you run out (shown greyed out so you know to restock!)
- **Full Customization**:
  - Adjustable icon size and padding
  - Configure rows and columns
  - Multiple anchor points (corners)
  - Frame strata control (display over other UI elements)
- **Drag & Drop Ordering** - Arrange categories and items in any order you want
- **Edit Mode Integration** - Move bars using WoW's built-in Edit Mode
- **Cooldown Tracking** - See cooldowns directly on your buttons
- **Item Counts** - Shows stack counts for consumables

## Usage

1. Type `/eb` or `/extrabars` to open the configuration panel
2. Click "+ Add New Bar" to create your first bar
3. Select categories or browse your inventory to add items
4. Use the Order tab to arrange items exactly how you want
